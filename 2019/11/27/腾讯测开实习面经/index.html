腾讯测开实习面经

发表于 2019-11-27 | 更新于 2019-12-01

其实这是一次意料之外的面试……因为我没有往腾讯投简历，但是受同学邀请也可以直接去现场面，还是有不少收获

一面

  • 自我介绍
  • 开始问项目，问得很细
    • 主要负责哪些模块的开发，用到了哪些框架
    • 针对管理员模块开始深入
      • DB是怎么设计的，整个系统的DB都是你设计的吗
      • 管理员表和酒店表改成一对多
      • 继续上面，改成多对多
      • …
      • 还有一些不记得了
    • 除了Java还有其他熟悉的语言吗
  • 一面过（据二面面试官说一面评价挺高）

二面

  • 面试官进来先询问我个人情况，是不是同学介绍来的？什么专业的？怎么理解你的专业互联网+？
  • 询问学校相关课程，有无互联网+特色课程？（都不想提创投和电路了……）
  • 重点来了，有哪些选修课程你觉得挺重要的？
    • 答曰：计组，计网，C++，UNIX
  • 你觉得你最熟悉哪个？
    • 实在不知道说啥，答了计网，自己坑自己（我计网不咋地）
  • 讲讲TCP的几种状态，再举例子
  • 又开始问项目，比刚刚更细了
    • 重新设计取消订单功能，表需要怎么改变
      • 在长时间的试探和面试官的引导下，过关
    • 增加修改订单功能，如何实现
      • 继续刚才的过程，过关
  • 终于不问项目了，开始问反射，动态反射和静态反射，反射的算法了解吗
    • 这里答的不好
  • Java内存模型，类在JVM中是怎样分配内存的，
  • 代码区溢出了怎么办
    • 我回答重新申请空间，但是不对
    • 问面试官，结果他也不知道……他猜测是抛出异常
  • 二面过，过几天会来电话三面
