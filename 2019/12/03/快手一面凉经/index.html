快手一面凉经

发表于 2019-12-03

面试快手的时候状态不是很好，比较被动，可能有第一次视频面的原因在里面，也暴露出自己对于技术的学习还不够深入的问题

  • 自我介绍
  • 介绍下项目，主要负责哪些模块，收获最大的是什么
  • MySQL索引是什么？MySQL的索引是如何实现的？
  • 你谈到了B+树，讲讲B+树吧
  • Spring AOP用到了什么设计模式？其底层实现能讲讲吗？
  • Spring的@Autowired和@Resource有什么区别？
  • HashMap的源码有研究吗？是如何实现的？
  • HashMap是如何处理哈希冲突的？
  • 讲一讲TCP四次挥手的过程
  • HTTP 400是什么状态？
  • 手撕算法：
    • 大小为2n的数组，其中有n个重复的数字，找出这个数字
    • 如何判断一棵树是平衡二叉树
    • 对一个有序数组进行二分查找，求找到的那个数第一次出现的位置

最后面试官看了我的博客，还给了建议，就是每一篇文章应该写的更深入一些，要花更多的时间去打磨一篇博文，这也是我将来需要不断改进的地方。

总的来说，面试过程虽然不尽如意，但还是有不少收获。
