京东Java开发实习面经

发表于 2019-12-04

共两面，都是电话面

一面

  • 自我介绍
  • 介绍 GC 相关算法，对 G1 了解吗？
  • LinkedList 和 ArrayList的区别？
  • Thread里start()和run()的区别？
  • 了解哪些设计模式？说一说最熟悉的一种设计模式，用生活中的场景举例。如何实现的？
  • static和final分别什么时候使用？
  • 如何判断一个链表有环？
  • 一个有序数组，下标为 0-98，但存放的数据为 0-99 范围内的整形，共 100 个，快速找出那个缺失的数字
  • 在学习 Spring 的过程中掌握了哪些知识？
  • 讲一讲AOP
  • Mybatis有和其它框架对比过吗？
  • 做项目的过程中遇到过什么问题？是怎么解决的？

二面

  • 自我介绍
  • 介绍项目中有哪些功能，主要负责哪些模块？详细讲讲你做的某个模块？
    • 推荐是如何实现的？
    • 搜索是如何实现的？
    • 为什么选择使用 Mybatis，而不用其它的持久层框架？
    • Mybatis如何处理参数比较多的情况下的多条件查询？如何接收这些参数？
    • 有考虑将查询条件封装吗？
    • Mybatis中#{}和${}的区别？
    • 有涉及到多表查询吗？
      • 在一条sql语句中实现多表查询？
      • JOIN和LEFT JOIN的区别？项目中有用到吗？这时候条件查询的关键字是什么？
    • 介绍下订单系统？
      • 如果订单状态变化的时候发生了异常，如何保证状态能正常切换或回滚？
    • 系统中还有哪些模块讲讲？
    • 项目采用的是什么设计架构？
    • 用户密码在数据库中是如何存储的？如果要加密的话该如何处理？
    • 项目开发过程中，使用git有遇到什么问题吗？
    • 你们是如何利用git进行版本控制的？
  • 了解 Spring Boot 中 JDK1.8 的特性吗？它有什么优点？
  • 对Java的面向对象是如何理解的？
  • 接口和抽象类的区别？
  • StringBuilder和StringBuffer的区别？
  • 在你们的项目中有用到多线程吗？实现多线程的方式？runnable和callable的区别？
  • Spring的主要作用？
  • Spring注入有哪些方式呢？
  • Spring IOC有自动装配，byName和byType的区别？如果有多个相同的type会怎样？
  • Spring Boot中用到了MVC设计模式，讲一讲？
  • Spring MVC的整个响应处理流程讲一讲？
  • Spring中用到了哪些设计模式？
  • 讲一讲你比较熟悉的设计模式？工厂模式的思想？
  • 讲讲你比较熟悉的数据结构
  • 栈的特点，和队列的区别？
  • 常见的排序算法讲一讲？快排的思想是什么？
  • 用Java实现快排的话，你会用什么数据结构去实现呢？
  • 你平时用的比较多的集合是哪些？
  • Set的特点？和ArrayList, Map的区别？
  • 什么样的数据是由Map去存？
  • ArrayList和其它集合结构的区别？
  • 常见的异常对象？
  • 如果程序报了空指针异常，你会用什么方式去排查？

  • 手撕代码：

    • 两个长为n的字符串数组，其中存在大量相同的字符串，找到两个数组中相同的字符串并打印出来

  • 后面电话掉线了……就没录了，不太记得问了啥

  • 平时有研究过代码规范吗？自己平时会注重这方面吗？

总结

整个面试不算难，一面比较注重基础，二面项目部分问得比较多，穿插了对框架、数据库等的考察。

最后面试官也给了我宝贵的建议，希望我在今后学习的过程中能多深入探究，比如说研究框架源码之类的。

几天后顺利拿到了京东的offer，感谢京东对我的认可！
