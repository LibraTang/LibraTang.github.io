Java对象引用处理机制

2018-09-24

其实就是多态，多个引用变量可以指向同一个物理对象，这些引用变量可以是不同的类型。

运行结果：

在内存中，pp和ss都指向了同样的地址

Java在运行时才处理别名引用，在程序运行过程中，虚拟机发现数组pp的第一个元素是S类型的对象,而不是P类型的。

在实际运行过程中，多态根据实际对象类型决定调用父类还是子类方法，而不是根据引用类型
